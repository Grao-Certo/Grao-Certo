# 📦 Grão Certo - Gestão e Controle de Silos

## 📖 Descrição
Projeto acadêmico desenvolvido para monitoramento do nível de grãos em silos agrícolas, utilizando sensores e um sistema web para visualização dos dados em tempo real.

A solução busca melhorar o controle de estoque, reduzir perdas e auxiliar na tomada de decisão do produtor.

---

## 🎯 Objetivo
Desenvolver um sistema de baixo custo capaz de:
- Medir o nível de grãos no silo  
- Estimar o volume armazenado  
- Exibir dados em tempo real em uma dashboard  

---

## ⚙️ Funcionalidades
- Monitoramento do nível de grãos  
- Cálculo de volume do silo  
- Dashboard com gráficos dinâmicos  
- Registro de dados em banco  
- Alertas para níveis críticos  
- Controle de acesso ao sistema  

---

## 🛠️ Tecnologias Utilizadas
- Arduino (sistema embarcado)  
- Dat-Acqu-ino (aquisição de dados do Arduino)  
- Sensores ultrassônicos  
- API para envio de dados  
- Banco de dados  
- HTML, CSS e JavaScript  
- Chart.js (visualização de gráficos)  
- Web Data Viz (análise e visualização de dados)  

---

## 📊 Visualização de Dados
A dashboard utiliza:
- **Chart.js** para criação de gráficos (linha, barra, etc.)  
- **Web Data Viz** para análise e exibição dos dados em tempo real  

---

## 📊 Resultados Esperados
- Redução de perdas no armazenamento  
- Melhor controle de estoque  
- Dados em tempo real  
- Apoio na tomada de decisão
